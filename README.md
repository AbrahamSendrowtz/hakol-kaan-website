# Hakol Kaan Website

This folder is ready for GitHub plus Cloudflare Pages.

## Publish

1. Push this `Website` folder to GitHub.
2. In Cloudflare Pages, create a project from the GitHub repo.
3. Set the build command to blank.
4. Set the output directory to `public`.
5. Deploy.

## SMS Number

The current site uses the public campaign number `+1 (929) 297-9785`.
If your Telnyx campaign uses a different public number later, update it in the HTML files before submitting the campaign.
